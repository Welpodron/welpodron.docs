## Подключение 

В случае если модуль поддерживает несколько способов подключения, используйте только тот, который больше подходит для проекта

<Alert>
Перед подключением модуля убедитесь, что все необходимые для требуемого модуля зависимости установлены и подключены!
</Alert> 

### Используя Bitrix 

Для подключения модуля используется метод <a target="_blank" href="https://dev.1c-bitrix.ru/api_d7/bitrix/main/loader/includemodule.php">`Loader::includeModule`</a>  

```php
Bitrix\Main\Loader::includeModule('welpodron.form');
```
